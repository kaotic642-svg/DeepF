Quick Start

Run order
  1. Start MediaMTX.
  2. В OBS указать Stream Server: rtmp://.../live.
  3. Stream Key: test.
  4. Запустить Start Streaming в OBS.
  5. Проверить, что OBS Input = Online.
  6. Загрузить Source Face и нажать Check Face.
  7. Нажать Start Processing.
  8. В OBS Media Source открыть rtmp://.../live/processed.
  9. Для OBS Virtual Camera использовать сцену с live/processed.
Не отправляйте сцену с live/processed обратно на RunPod, иначе получится видеопетля. Для теста лучше: OBS #1 отправляет webcam, OBS #2 или отдельная сцена принимает live/processed.
